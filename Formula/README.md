# Homebrew Tap for ᜀᜎᜀᜎ (alaala)

Homebrew formulae for [alaala](https://github.com/0xGurg/alaala) - A semantic memory system for AI assistants.

## Installation

```bash
brew tap 0xGurg/alaala
brew install alaala
```

## Usage

```bash
# Initialize project
alaala init

# Start MCP server
alaala serve

# Show version
alaala version
```

## Uninstallation

```bash
brew uninstall alaala
brew untap 0xGurg/alaala
```

To also remove data and configuration:
```bash
rm -rf ~/.alaala
```
